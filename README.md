# Algorithm for automatic metadata extraction

This CLI tool is to make routine tasks of extracting metadata from State/GOST 2.104-2006 21.101.2020 standard jpg/png/pdf raster image for convenient way of document management in digital archive systems. Specifically useful for old archive images that are preserved through legacy and are hard to localize/classify by hand due to their amounts.

> **Note:** This project was completed as part of a course requirement.
